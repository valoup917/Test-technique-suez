Lead développeur full stack


En tant que lead développeur full stack, j'ai dirigé une équipe de 5 personnes pour un projet passionnant visant à automatiser des tâches en interrogeant plusieurs API ouvertes telles que Spotify, YouTube, Instagram et la météo. Nous avons développé une application mobile en utilisant Flutter pour le front-end et j'ai également conçu l'architecture du back-end, en utilisant des microservices avec une API Gateway que j'ai développée à partir de zéro en utilisant JavaScript. Le projet était intéressant car il a impliqué la gestion d'une petite équipe et un défi technique en utilisant GRPC. Il a également été reconnu par l'école d'informatique Epitech comme un très bon projet. Les technologies utilisées dans ce projet comprenaient JavaScript, React JS, Flutter, PostgreSQL, Python Flask, Python Django, Go et un cloud avec Scaleway. Ce projet a été une expérience très enrichissante pour moi en termes de gestion d'équipe et de développement de compétences techniques.
